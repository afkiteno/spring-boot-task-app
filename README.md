# Spring Boot Task Management API (v2)

A backend API for managing tasks built using Spring Boot. This project was created to learn about layered architecture, Data Transfer Objects (DTOs), object mapping, and global exception handling.

Docker & Docker Compose – Used to containerize and run the pre-built Frontend UI.

## Tech Stack & Tools
* Java 17 / Spring Boot 4 (Web, Data JPA, Validation,H2 Database)
* Docker & Docker Compose (Run Frontend UI)
